Inclusive AI Assessment Cloud (IAAC)

Requirements

#Elevator Pitch

An AI‑powered AWS platform that converts one exam into multiple personalized experiences so differently‑abled candidates can be evaluated fairly without manual accommodations.

#Problem

Existing online exam systems assume every candidate interacts the same way. Candidates with visual, motor, or cognitive impairments require human scribes, extra approvals, or separate infrastructure. This makes large‑scale accessible testing expensive, inconsistent, and unfair.

#Solution Overview

IAAC dynamically adapts the exam delivery and answer capture mode in real‑time using AI services. The platform personalizes interface, timing, and interaction method per candidate session.

#Target Impact

Inclusive government recruitment

Accessible university exams

Fair corporate hiring assessments

#MVP Features (Hackathon Scope)

Accessibility onboarding questionnaire

Adaptive UI (contrast, layout, font, navigation)

Text‑to‑speech question narration

Speech‑to‑text answer submission

Automatic extra‑time adjustment

Pause & resume reliability

Fair scoring normalization

#User Flow

Register → Accessibility Profiling → Personalized Exam → Multimodal Answer → Normalized Result

#Success Criteria

Candidate completes exam without human assistance

Same question usable across multiple disability types

Consistent score distribution independent of interaction mode

#What Makes This Unique

Instead of separate accessible exams, the same exam intelligently reshapes itself per user — scalable accessibility rather than manual accommodation.
